# PPL - Sistem Manajemen Perpustakaan

Repository ini berisi perancangan perangkat lunak Sistem Manajemen Perpustakaan berbasis web.
Sistem ini dirancang untuk membantu pengelolaan data buku, data anggota, transaksi peminjaman,
pengembalian, serta pembuatan laporan perpustakaan secara cepat dan akurat.

## Nama Aplikasi
Sistem Manajemen Perpustakaan

## Studi Kasus
Perpustakaan masih menggunakan sistem manual dalam pencatatan buku, anggota, serta transaksi peminjaman dan pengembalian.
Hal tersebut menyebabkan proses pencarian buku menjadi lambat, sering terjadi kesalahan pencatatan, serta kesulitan dalam pembuatan laporan.

## Tujuan Pengembangan Sistem
- Mempermudah pengelolaan data buku dan anggota
- Mempercepat proses peminjaman dan pengembalian buku
- Mengurangi risiko kehilangan data
- Membuat laporan perpustakaan secara otomatis dan akurat

## Aktor Sistem
- Admin / Pustakawan
- Anggota Perpustakaan

## Fitur Utama Sistem
- Login dan Logout
- CRUD Data Buku
- CRUD Data Anggota
- Transaksi Peminjaman Buku
- Transaksi Pengembalian Buku + Perhitungan Denda
- Laporan Peminjaman dan Pengembalian
